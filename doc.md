###   引用Vant组件库步骤


步骤一 通过 npm 安装
需要注意的是 package.json 和 node_modules 必须在 miniprogram 目录下
在miniprogram文件夹下初始化npm(npm init)

# 通过 npm 安装
npm i @vant/weapp -S --production

# 通过 yarn 安装
yarn add @vant/weapp --production

# 安装 0.x 版本
npm i vant-weapp -S --production


# 开发者工具栏-->工具-->构建npm（此时miniprogram下就会有一个）
